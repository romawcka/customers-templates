# customers-templates

Standalone production-ready React package for the Rolling Offer feature.

## Features

- React 18+ compatible
- Next.js App Router / SSR compatible
- Client-side interactive component with proper `"use client"`
- Dynamic props for products, image URLs, badges, quantities, CTA text, and timing
- Vite library build
- TypeScript declarations
- ESLint, Prettier, Storybook

## Install locally

```bash
npm install
npm run build
npm run storybook
```
